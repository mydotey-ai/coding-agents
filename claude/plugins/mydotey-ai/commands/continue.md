# continue

AI 任务执行中途停下来时，用此命令让它继续完成未完成的工作。

## 执行步骤

1. **回顾上下文**
   - 回顾当前对话中用户最初的任务请求
   - 回顾已完成的工作（已修改的文件、已执行的命令、已产生的输出）
   - 识别任务的当前进展状态

2. **确定剩余工作**
   - 对比最初任务目标和已完成部分
   - 列出尚未完成的具体步骤
   - 如果有 task list，检查未完成的 task

3. **继续执行**
   - 从中断的地方继续，不要重复已完成的工作
   - 不要重新询问已经明确的需求
   - 不要等待确认，直接继续干活
   - 保持与之前一致的实现风格和方案

## 注意事项

- 不要解释为什么之前停了，直接继续干
- 不要输出冗长的进度回顾，简短说明当前在做什么即可
- 如果原始任务已经全部完成，告知用户即可
